时间管理

核心原则

  • 凌晨2点强制昏迷，扣钱；1点前回家最安全
  • 12点后体力消耗加快
  • 每日优先级：浇水 → 收获 → 动物 → 其他
  • 尽快获得洒水器，节省大量时间
  • 每天看电视：运势（影响矿洞运气）、食谱、生活提示

工具升级顺序

优先级工具原因
1水壶浇水是最耗时的每日任务
2斧头砍硬木需要铜斧以上
3镐打破大石头需要钢镐
4锄头后期大面积翻地有用

提示：升级需2天，选择不需要该工具的日子（如下雨天升级水壶）

加工赚钱

设备输入 → 输出利润提升
酿酒桶水果 → 酒3倍售价
保存罐蔬菜 → 腌菜2倍+50g
蛋黄酱机蛋 → 蛋黄酱利润翻倍
奶酪压制机奶 → 奶酪利润翻倍
油制造机松露 → 松露油大幅增值

最赚钱加工品

  • 星之果实酒 = 2,250g（原果实750g）
  • 远古水果酒 = 1,650g+
  • 山羊奶酪（金星）= 400g

每日清单

建议每日流程

  1. 看电视（运势 + 今日食谱）
  2. 浇水 / 检查洒水器区域
  3. 收获成熟作物
  4. 抚摸动物 + 收产品
  5. 检查加工设备
  6. 查看信箱
  7. 剩余时间：矿洞 / 钓鱼 / 社交
